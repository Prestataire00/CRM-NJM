# 🚀 Mettre votre projet sur GitHub

**Bonne nouvelle !** J'ai déjà tout configuré sur votre ordinateur (Git initialisé + lien vers GitHub enregistré).

Il reste **une seule étape manuelle** car votre token ne m'a pas autorisé à créer le dépôt automatiquement.

## Étape 1 : Créer le dépôt sur GitHub

1.  Allez sur ce lien direct : [Créer un nouveau dépôt](https://github.com/new).
2.  Nom du dépôt : **`crm-formation-supabase`** (Exactement ce nom).
3.  Visibilité : **Private** (Privé).
4.  **Important** : Ne cochez RIEN d'autre (pas de README, pas de .gitignore). Le dépôt doit être vide.
5.  Cliquez sur **Create repository**.

## Étape 2 : Envoyer le code

Une fois le dépôt créé, revenez dans votre terminal (VS Code) et lancez simplement cette commande :

```bash
git push -u origin main
```

Votre code sera alors envoyé ! 🎉
